# Exemple : Site web simple avec MySQL/MariaDB

En plus des fonctionnalités de l'exemple
[_"Site web simple avec SQLite"_](../01-site-web-simple-avec-sqlite/README.md),
cet exemple illustre la création d'une application web simple en PHP avec :
